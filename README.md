## 🏃 How to Run

### Prerequisites
- Java Development Kit (JDK) 8 or higher installed
- Command line or terminal access

### Running a Program

1. **Navigate to the Java directory:**
   ```bash
   cd path/to/Java
   ```

2. **Compile and run any Java file:**
   ```bash
   java filename.java
   ```
   
   Example:
   ```bash
   java Threads.java
   java Array.java
   java hello.java
   ```

3. **For programs using packages (like Packages.java):**
   ```bash
   java Packages.java
   ```
